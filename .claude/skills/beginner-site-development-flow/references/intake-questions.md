# Intake questions reference

初心者に聞く質問は、最初にまとめて聞きすぎない。基本は次の4つだけ。

## 必ず聞くこと

1. サイト名は何にしますか？
   - `SITE_TITLE` に使う。
   - 例: `山田太郎のブログ`, `Libe Hair Salon`, `小さな雑貨店サンプル`

2. どんなサイトにしますか？
   - 回答から `SITE_DESCRIPTION` を作る。
   - 例: `日々の学びを記録する個人ブログ`, `店舗情報とお知らせを載せる美容室サイト`

3. 参考となるデザインはありますか？
   - URL、画像、既存サイト、スクリーンショットなど。
   - ない場合は「シンプル」「やさしい」「信頼感」などの雰囲気を聞く。

4. 作成したいページやセクションを教えてください。
   - 例: トップ、プロフィール、サービス、料金、ブログ、お知らせ、よくある質問、アクセス

## 聞き方の例

```text
まずサイトの中身を決めましょう。わかる範囲で大丈夫です。

1. サイト名は何にしますか？
2. どんなサイトにしますか？（ここから説明文を作ります）
3. 参考にしたいデザインはありますか？URLや画像があればください。
4. 作りたいページやセクションを教えてください。
```

## 注意

- 住所、電話番号、個人情報などを無理に聞かない。
- 問い合わせフォームやログイン機能は安易に提案しない。
- Cloudflare の公開URLや Web Analytics は、この段階では聞かない。
